# Favicon Files

Place your favicon files in this directory:

## Required Files (at least one):

- **favicon.ico** - Standard favicon (16x16 or 32x32)
- **favicon.png** - PNG favicon (32x32 recommended)
- **favicon.svg** - SVG favicon (scalable, modern browsers)

## Optional Files:

- **apple-touch-icon.png** - Apple touch icon (180x180)
- **site.webmanifest** - Web app manifest file

## File Formats:

- **favicon.ico** - Traditional format, works everywhere
- **favicon.png** - Modern PNG format
- **favicon.svg** - Scalable vector, best quality
- **apple-touch-icon.png** - For iOS home screen

## Notes:

- The theme will automatically detect and include any files that exist
- You don't need all formats - use what you have
- SVG is recommended for modern browsers
- ICO is recommended for maximum compatibility

## Example:

```
imgs/favicons/
  ├── favicon.ico
  ├── favicon.png
  ├── favicon.svg
  └── apple-touch-icon.png
```
